# 公開展覽

當你完成展間想要讓其他人來逛時，你可以點擊「公開/不公開」按鍵允許訪客將參觀你的展覽。

![](<../../.gitbook/assets/截圖 2022-04-08 下午1.58.37.png>)

如果你暫時不希望被看到，也可以藉此設定為不公開，「公開/不公開」的狀態會直接顯示在圖示上。

![](<../../.gitbook/assets/截圖 2022-04-08 下午2.19.30.png>)
